セレクターの組み合わせ

セレクターを組み合わせる書き方

h2,.abc

h2と.abcに同じスタイルを適用させる
カンマ区切りでいくつでも指定できる

p.def

p要素であり、.defクラスという、両方のセレクターを同時に満たす要素にスタイルを適用させる
くっつけて書く書き方の場合、要素型セレクター同士をくっつけることはできない

main > h1

ここは対象内

ここは対象外

main の直下の階層の h1 という意味になる。直下以外の階層は対象外。

header h1

ここも対象内

ここも対象内

header の下の階層すべての h1 という意味になる。

p + p

p1　同じ階層の中で、pの直後にあるpという意味になります。

p2　このように2行だけ下線を引きたいなどといった場合に使えます。

p3　隣接した要素の間に何らかのスタイルを差し込みたい場合に使えます。
